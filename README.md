# イラン戦況 × ドル円 モニター（Iran Briefing）

毎朝のイラン情勢をリアルタイムに収集し、**ドル円（USD/JPY）相場への影響度**とともに報告するパーソナル・インテリジェンス・ダッシュボード。

---

## 概要

Claude API（claude-sonnet）のWeb検索機能を活用し、世界15社以上のメディアからイラン関連ニュースを自動収集・要約します。FXトレーダー向けに地政学リスクを「為替直結」視点で整理した、個人利用向けPWAアプリです。

---

## 目的

- イラン情勢（軍事・外交・エネルギー）がドル円に与えるリスクを毎朝把握する
- ホルムズ海峡・原油価格・制裁動向など、FXに直結する情報を優先表示する
- モバイルからワンタップで最新ブリーフィングを取得できる環境を整える

---

## 主な機能

| 機能 | 内容 |
|------|------|
| FXリスク判定 | HIGH / MEDIUM / LOW の3段階でドル円影響度を表示 |
| 多ソース収集 | CNN, BBC, Reuters, Bloomberg, Al Jazeera, Lloyd's List 等15社以上 |
| セクション分類 | 為替直結 / 戦況 / 外交 / イラン側 / 国際反応 |
| PWA対応 | スマートフォンのホーム画面に追加してアプリとして利用可能 |
| 日本語UI | 全表示を日本語で統一 |

---

## 使い方

1. GitHub Pages でホスティングされた `index.html` をブラウザで開く
2. 「📡 最新ブリーフィングを取得」ボタンを押す
3. 15〜30秒でブリーフィングが生成される

> **Note:** Claude API キーはブラウザ側で直接呼び出す構成のため、個人利用を前提としています。
